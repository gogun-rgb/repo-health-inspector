# Changelog

All notable changes to repo-health-inspector are documented here.

## 0.1.0

- Implemented the Typer-based `repo-health` CLI.
- Added remote GitHub repository inspection with safe archive download and static analysis.
- Added local repository inspection without network access.
- Added transparent rule-based scoring across documentation, testing, structure, security, and maintenance.
- Added Markdown and JSON report generation.
- Added pytest coverage, Ruff linting and formatting, mypy type checking, and GitHub Actions CI.
